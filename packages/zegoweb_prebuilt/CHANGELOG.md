## 0.0.1

- Initial release.
- Dart wrapper for `@zegocloud/zego-uikit-prebuilt` JS UIKit via `dart:js_interop`.
- `ZegoPrebuilt` — create instance, join room, listen to events, destroy.
- `ZegoPrebuiltView` — single Flutter widget rendering the prebuilt call UI.
- `ZegoPrebuiltConfig` — typed config with raw override support.
- UMD script-tag SDK loader.
- Test kit token generation for development.
