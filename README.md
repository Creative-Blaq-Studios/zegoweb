# zegoweb

Unofficial community Flutter web plugin family wrapping ZEGOCLOUD's Express
Video Web SDK. Not affiliated with or endorsed by ZEGOCLOUD.

This monorepo hosts three packages:

| Package            | Status  |
| ------------------ | ------- |
| `zegoweb`          | In progress |
| `zegoweb_prebuilt` | Planned |
| `zegoweb_ui`       | Planned |

See `docs/superpowers/specs/` for design specs.
